# Tower Defense

A simple tower defense game built with Godot 4.3.

## How to Run

1. Download [Godot 4.3](https://godotengine.org/download) (standard version)
2. Clone this repository
3. Open Godot and import the project folder
4. Press F5 to play

## Gameplay

- **Place towers** by clicking on green cells
- **3 tower types:** Basic ($50), Sniper ($100), Rapid ($75)
- **Survive 10 waves** to win, then continue to endless mode
- **Earn gold** by defeating enemies

## Controls

- Click tower buttons in footer to select tower type
- Click on the map to place towers
- Click "Start Wave" to begin the next wave early
